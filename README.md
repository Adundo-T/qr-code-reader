# QR Code Reader

A web app that scans QR codes using your device camera.  
Built with **HTML, CSS, JavaScript**.  

## 🚀 Features
- Real-time QR code scanning  
- Camera switching (front/back)  
- Pause/Resume scan  
- Scan history with timestamps  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript
